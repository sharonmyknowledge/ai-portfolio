# AI Portfolio

Sharon Mathis — AI Portfolio site.

This repository contains a single-page static site (index.html) intended to be published with GitHub Pages at:

https://sharonmyknowledge.github.io/ai-portfolio

Deployment

1. The site is already committed to the `main` branch. Enable GitHub Pages in repository Settings → Pages → Source: `main` branch, folder `/`.
2. Wait a minute for the site to deploy. GitHub Pages will provide the exact URL in the Pages settings.

Files

- index.html — the full site.

Customizations

- To change your LinkedIn, website, or email update the links in index.html.
- To change the accent color edit the CSS variables at the top of the file.
